HTML
HTMl là chữ viết tắt của cụm từ HyperText Markup Languge (ngôn ngữ đánh dấu siêu văn bản).
CSS
CSS là viết tắt cụm từ Cascading Style Sheet, nó là một ngôn ngữ quy định cash hiển thị của các thẻ html trên trang web.
Javascript
JavaScript là một ngôn ngữ lập trình kich bản dựa trên đối tượng được phát triển từ các ý niệm nguyên mẫu.
